# Glyph Run — audio credits

All the guzheng in the game is built from one clean source note:

- **guzheng-bgm.mp3** — the looping background bed, rendered by pitch-shifting and
  sequencing that note into a calm pentatonic pattern.
- **guzheng-pluck-a.mp3 / guzheng-pluck-b.mp3** — the two real plucks, used for the
  character / weapon selection tones (pitch-shifted per choice).

Source note: **"guzheng-instrument-single-note-sound" by nanliu_music**
- https://freesound.org/s/847157/
- License: **Creative Commons 0 (CC0 / public domain)** — no attribution required;
  credited here and in-game as good practice.

The bed slows its tempo (pitch held) when you land hits and faints when you take
one. 编钟 bells and all other SFX are synthesized in-code (no license needed).
